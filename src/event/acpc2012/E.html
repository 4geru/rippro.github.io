問題概要

入力

  • HTMLのようなファイル
  • JavaScriptのようなファイル
  • ユーザの情報

出力

  • ユーザの最終的な画面の様子
  • ユーザの人数分

例

index.dml 
<dml>
	<script>s</script>

</dml>
s.ds 
on {
	fade.visible = true;
}
off {
	fade.visible != true;
}

幅(w)：15，高さ(h)：3
index.dmlから表示
入力: なし

...............

on off.........

struct Line {
	vector<Node*> index;
	bool value;
};

  • 再帰的構文解析 + 深さ優先探索
  • splitがあると楽かも
    • Java: String.split()
    • C: strtokは使い方に注意
